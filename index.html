このアプリについて（v9.2）

  • 位置は 編集版（出題に使用）と WHO要約 の二層管理。
  • 有効/無効で出題対象を制御、編集済フラグで校正進捗を可視化。
  • CSV/JSON：旧「位置」列も自動的に編集版へ取り込み互換。
  • コード順：CV → GV → LU → LI → ST → SP → HT → SI → BL → KI → PC → SJ → GB → LV。

初期サンプルはWHO要約の一部10穴です。授業配布のCSVテキストを「一括貼り付け」から取り込んでご利用ください。
